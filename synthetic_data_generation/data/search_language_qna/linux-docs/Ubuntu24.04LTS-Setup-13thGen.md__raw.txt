Question: How to disable power save for Intel Wi-Fi?
Answer: To disable power save for Intel Wi-Fi, run the command 'sudo sed -i '/^wifi.powersave = 3$/s/3/2/' /etc/NetworkManager/conf.d/default-wifi-powersave-on.conf' after checking if power save is on with 'iw dev $interface get power_save'.
