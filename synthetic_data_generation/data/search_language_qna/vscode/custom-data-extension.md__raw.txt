Question: custom data html support
Answer: Custom Data format allows extension authors to extend VS Code's HTML language support.

Question: custom data css entities
Answer: Custom Data format provides language support for CSS entities defined in custom data files.

Question: vscode custom data package json
Answer: The `package.json` of a VS Code extension should include `contributes.html.customData` and `contributes.css.customData` to use custom data.

Question: custom data auto-completion
Answer: Custom Data format provides auto-completion and hover information for HTML/CSS entities defined in custom data files.

Question: custom-data-sample location
Answer: The custom-data-sample can be found in [microsoft/vscode-extension-samples](https://github.com/microsoft/vscode-extension-samples).
